# Message-Cleaner---Plugin-Vencord

Message Cleaner est un plugin qui permet de supprimer rapidement et en masse plusieurs messages, ou l’intégralité des messages, dans la conversation actuellement ouverte, en un seul clic.

Message Cleaner is a plugin that allows you to quickly bulk delete multiple messages, or all messages, from the currently open conversation with a single click.
